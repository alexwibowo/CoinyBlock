# CoinyBlock
CoinyBlock iOS retro clicker game

![logo](https://github.com/chriscomeau/Portfolio/blob/master/images/coinyblock_icon.jpg)

##### Notes

* Skyriser Media, 2017
* Objective-C / Swift
* SpriteKit, UIKit, CoreAnimation
* TodayWidget extension
* StickerPack extension
